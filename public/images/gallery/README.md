# Gallery Images

Drop your car and work photos here. These will be used directly on the website.

## Folder structure (optional but helpful)

You can organize photos into subfolders:

- `before-after/` — before & after transformation shots
- `detailing/` — detailing work close-ups
- `ceramic-coating/` — ceramic coating jobs
- `paint-correction/` — paint correction work
- `vehicles/` — finished vehicle showcase shots

## Tips for best results on the website

- Use high-resolution photos (at least 1080px wide)
- JPG or PNG format works best
- Landscape orientation works better for most layouts
- Name files descriptively (e.g. `black-bmw-ceramic-coat.jpg`) so they're easy to reference
