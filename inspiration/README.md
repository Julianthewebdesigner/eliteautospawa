# Website Inspiration

Drop any reference photos, screenshots, or mockups here that show how you want the website to look and feel.

## Suggestions for what to include

- Screenshots of other auto spa / detailing websites you like
- Color palette examples
- Font or typography examples
- Layout ideas (hero sections, service cards, galleries, etc.)
- Logo or branding assets
- Any photos of your own shop or vehicles to use on the site

The more examples you share, the closer the final build will match your vision.
